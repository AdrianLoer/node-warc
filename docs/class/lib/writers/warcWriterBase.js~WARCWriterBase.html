    public constructor(noHTTP2: boolean) source

    Create a new WARCWriter

    Params:

    NameTypeAttributeDescription
    noHTTP2 boolean
    • optional
    • default: false

    noHTTP2 - The web uses HTTP/2 but all known replay systems can only replay HTTP <= 1.1. But if you turn HTTP/2 into HTTP/1.1 it will replay for the most part. True means modify the HTTP headers False means leave the HTTP headers as they occurred
