    WARCRecorderBuilder

    Builds WARC Records by keeping track internally of the current WARC Record WARC-TYPE being parsed by a parser e.g. WARCGzParser

    public clear() source

    Clears the builders internals. Do not call this functions unless parsing has completely finished this will completely remove all information pertaining to building of the current WARC Record. This method is intended to be used by the WARC parsers.
