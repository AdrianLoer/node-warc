    public fieldIdentifiers: {begin: Buffer, info: Buffer, mdata: Buffer, req: Buffer, res: Buffer, revisit: Buffer, crlf: Buffer, empty: Buffer} source

    Buffers from hex strings representing the a WARC records WARC-TYPE or record/record-section separator as seen by the parser. Due to the large size of WARC files and the binary data contained within it is more efficient to parse the files contents looking for these sentinels than converting the buffer to an UTF-8 string first
